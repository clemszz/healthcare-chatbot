# 🩺 Chatbot Santé Travail

## Objectif
Ce projet est un chatbot d’orientation en santé au travail.

Il permet d’aider les utilisateurs à :
- Prendre un rendez-vous
- Comprendre un problème de santé lié au travail
- Gérer le stress professionnel
- Demander un aménagement de poste

---

## Fonctionnement

Le chatbot utilise :
- une logique de détection d’intention (NLP simple)
- une interface interactive avec Streamlit
- un système de réponses guidées
